# NFL Pass Coverage Dashboard

An interactive dashboard built with NFL Big Data Bowl 2024 data. Filter 18,000+ plays by coverage type, down, distance, formation, and more.

## Files

- `index.html` — Main page
- `style.css` — Styles (dark theme, glassmorphism)
- `script.js` — Filter logic + Chart.js visualizations
- `data.json` — 18,009 plays converted from NFL Big Data Bowl CSV

## Deploy to Vercel (Browser Only)

1. Go to github.com/new and create a repo (e.g. `nfl-pass-dashboard`)
2. Drag and drop all 4 files into the repo
3. Go to vercel.com → Add New Project → Import your GitHub repo
4. Hit Deploy — done!
